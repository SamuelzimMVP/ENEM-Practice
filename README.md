# ⚡ ENEM Speedrun - Frontend 🎨

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Interface moderna e responsiva do **ENEM Speedrun**. O frontend foi desenhado para ser rápido, intuitivo e focado na performance do estudante sob pressão.

## 🚀 Sobre o Projeto
O ENEM Speedrun é uma plataforma onde o tempo é seu maior adversário. Diferente de simulados tradicionais, aqui o foco é a agilidade mental. O site permite que você:
1. Escolha sua área de maior afinidade.
2. Defina o tamanho do desafio.
3. Responda questões reais do ENEM com cronômetro ativo.
4. Desbloqueie conquistas exclusivas baseadas no seu desempenho.

## ✨ Funcionalidades Visuais

- **🥈 Galeria de Badges**: Página inicial com exibição dinâmica de conquistas (Bronze, Prata, Ouro, Platina e Diamante).
- **🕹️ Interface de Quiz**: Layout limpo que minimiza distrações, com barra de progresso visual e cronômetro em tempo real.
- **📈 Pódio Interativo**: Ranking nacional com destaques para o Top 3 em cada categoria.
- **🔔 Notificações Automáticas**: Alertas estilo Steam que aparecem instantaneamente ao desbloquear uma nova conquista.
- **🔄 Session Recovery**: Graças ao sistema de Refresh Token, o frontend mantém você logado de forma contínua durante suas maratonas de estudo.

## 🛠️ Tecnologias Decorativas

- **CSS Vanilla**: Estilização purista focada em performance e carregamento instantâneo.
- **Design Gov.br**: Identidade visual inspirada nos padrões oficiais para transmitir confiança e seriedade.
- **Animações Sugestivas**: Micro-interações que tornam a experiência de gamificação mais prazerosa.

## 🏁 Como Rodar

1. Clone o repositório.
2. Certifique-se de que a `API_BASE` em `assets/js/config.js` aponta para `http://localhost:3001` (para testes locais).
3. Abra o `index.html` usando um servidor local (como a extensão Live Server do VSCode ou `python -m http.server`).
